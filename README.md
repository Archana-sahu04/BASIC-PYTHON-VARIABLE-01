# BASIC-PYTHON
Python is a high-level,easy-to-learn programming language used for web development,data science, automation, AI, support oops concept and more.
It is focused on readability and simplicity,making it beginner-friendly.
It developed by GUIDO VAN ROSSUM in 1991 feb 20 at NRI(Netherland)
Python came from fun tv show called"Complete monty python's flying circus"-broadcast in BBC channel.
It is dynamic programming laguage because it dosenot declare any datatypes. and it is freeware and opensource.
It contains rich library like-NUMPY,PANDAS,MATPLOTLIB etc
Some flavours of python are -Cpython,Jpython,Ironpython,Rubypython,Anacondapython.
Both compiler and interpreter are used to convert a program wriiten in high-level language into machine code.
